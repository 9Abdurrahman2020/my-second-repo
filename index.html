I Can push code to github now

First time we need 6 steps to push code to github:

  1. git init
  2. git add .
  3. git commit -m "git massage"
  4. git branch -M main
  5. git remote add origin repository location
  6. git push -u origin main

Second time we need 3 step to push code to github:

  1. git add .
  2. git commit -m "commit here"
  3. git push
